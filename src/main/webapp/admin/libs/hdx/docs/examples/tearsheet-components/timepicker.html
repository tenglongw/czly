Example Description Usage
This class is a widget for selecting a time. We've added a thin layer around bootstrap-timepicker. 
<div class="input-append bootstrap-timepicker">
  <input type="text" class="form-control input-sm timepicker">
  <span class="input-group-btn">
		<span class="add-on">
	    <button class="btn btn-default" type="button">
	      <i class="icon-time"></i>
	    </button>
		</span>
  </span>
</div>

<script type="text/javascript">
    require.config({
    baseUrl: '../src'
  });

  require([
    'jquery',
    'timepicker'
  ], function ($) {
    $('.timepicker').timepicker();
  });
</script>
Script Requirements:
  • bootstap-timepicker
  • bootstrap
  • jquery, require
  • timepicker
