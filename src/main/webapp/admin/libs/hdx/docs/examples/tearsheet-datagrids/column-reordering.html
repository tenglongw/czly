Column reordering

A Datagrid using interactive column reordering.. Drag any column using the mouse, a blue line to the right of the nearest column will indicate where the dragged column will be placed when dropped.

Example
Usage

Markup:

<table cellpadding="0" cellspacing="0" border="0" class="table table-bordered" data-table-name="dt-col-reorder-example"></table>

Script:

//Column reordering
$('table[data-table-name="dt-col-reorder-example"]').basicdatagrid({
	'aoColumns': [
		{ 'sTitle': 'Rendering engine', 'mData': 'engine', 'sClass': 'essential'},
		{ 'sTitle': 'Browser', 'mData': 'browser', 'sClass': 'essential'},
		{ 'sTitle': 'Platform(s)', 'mData': 'platform', 'sClass': 'optional'},
		{ 'sTitle': 'Engine version', 'mData': 'version'},
		{ 'sTitle': 'CSS grade', 'mData': 'grade', 'sClass': 'essential'}
	],
	'useFloater': false,
	'sAjaxSource': '../assets/data/ao.browsers.json',
	'plugins': ['R'] //enable the col-reorder plugin (assumes 'col-reorder-amd' is on the page)
});

Notes:

  • We use the DataTables project's "ColReorder" plugin for this functionality.
  • You must include the plugin library on the page. The requirejs id for this in HDX is 'col-reorder-amd'
  • Configure according to the plug-in's documentation with the following changes.
    • Datatable "plugins" have been moved out of the "sDom" settings parameter into a separate "plugins" settings Array parameter, so enable the plugin by setting 'plugins': ["R"] on your table
Script Requirements:
  • datagrids
  • col-reorder-amd
