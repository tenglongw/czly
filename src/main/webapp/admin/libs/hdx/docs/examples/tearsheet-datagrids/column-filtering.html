Column filtering

A Datagrid using interactive column filtering.

Note: There is a bug with the 3rd party datatables component. On the iPad, when attaching a touch event to the body via the body tag or Javascript/jQuery, this text field input is blocked from allowing input
Example
Usage

Markup:

<table cellpadding="0" cellspacing="0" border="0" class="table table-bordered" data-table-name="dt-col-filter-example"></table>

Script:

//Filtering
$('table[data-table-name="dt-col-filter-example"]').basicdatagrid({
  'aoColumns': [
    {'sTitle': 'Rendering engine', 'mData': 'engine', 'sClass': 'essential'},
    {'sTitle': 'Browser', 'mData': 'browser', 'sClass': 'essential', 'filter':true},
    {'sTitle': 'Platform(s)', 'mData': 'platform', 'sClass': 'optional', 'filter': true},
    {'sTitle': 'Engine version', 'mData': 'version', 'filter':true, 'filterPlaceholder':'Filter by this field'},
    {'sTitle': 'CSS grade', 'mData': 'grade', 'sClass': 'essential', 'filter':true}
  ],
  'sAjaxSource': '../assets/data/ao.browsers.json',
  'useFloater': false,
  'isResponsive': true
});

Notes:

  • Use the "filter:true" and, optionally, "filterPlaceholder:..." members on each aoColumns element for the columns for which you want to enable column filtering..
Script Requirements:
  • datagrids
