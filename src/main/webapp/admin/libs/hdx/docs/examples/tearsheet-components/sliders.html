Examples






Usage

Markup:

<input type="hidden"
	class="slider col-md-8"
	value="50"
	data-min="0"
	data-max="100"
	data-slider-steps="10"
	data-slider-highlight="#0098d6"
	data-slider-label-position='outside'
	data-slider-snap='true'
	data-slider-label-suffix=" RPM"
	data-slider-button-position="apart"
	data-slider-tickmarks='4'
	data-slider-scale='5'
/>

Script:

$('.slider').slider({
});

Attributes:

Name Type Default Description
value integer Initial slider value
data-min integer Minimum value for slider
data-max integer Max value for slider
highlight string '#009ae1' Color code for highlighter portion of track
data-slider-buttons true|false true flag to specify if slider should have buttons to increment and decrement the value.
data-slider-button-position 'apart'|'together' 'apart' Determines if increment and decrement buttons should be next right and slift ends of slider or placed next to each other on the right side of the slider
data-slider-tickmarks number 0 Number of small ticks between two large ticks with number values
data-slider-scale number 0 Number of intervals with large tick marks
data-slider-label-position 'outside'|'inside' 'outside' Determines if label should be displayed outside the bordered area or inside the bordered area
data-slider-label-prefix String '' Label prefix
data-slider-label-suffix string '' Label suffix
data-slider-snap true|false false Determines if thumb will snap to closest discrete multiple of 'step' when thumb is adjusted manually.
data-slider-steps integer 0 Unit of change when + or - button is pressed.
Script requirements:
  • slider
