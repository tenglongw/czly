$('.mega-dropdown').each(
  function() {

 Basic Tabs

Markup:

<ul class="nav nav-tabs">
  <li class="active"><a href="#tab1" data-toggle="tab">Section 1</a></li>
  <li><a href="#tab2" data-toggle="tab">Section 2</a></li>
  <li><a href="#tab3" data-toggle="tab">Section 3</a></li>
</ul>

Activate tab navigation without javascript by specifying data-toggle="tab" on an element. Adding the nav and nav-tabs classes to the tab ul will apply the bootstrap tab styling.

To align nav links, use the .pull-left or .pull-right utility classes. Both classes will add a CSS float in the specified direction.

Script:

$('#myTab a').click(function (e) {
  e.preventDefault();
  $(this).tab('show');
})

To make tabs fade in, add .fade to each .tab-pane.

Variant positions can be implemented by adding .tabs-left, .tabs-right or .tabs-below to .tabbable.

For .tabs-below, .tab-content and .nav-tabs need to be in reverse order.

<div class="tabbable tabs-below">

    …
  </div>

  </ul>
