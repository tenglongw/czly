Column hiding

A Datagrid using interactive column hiding.

Example
Usage

Markup:

<div class="pull-right">
  <div class="col-vis-dropdown-container"></div>
  <div class="input-append">
    <input type="text" class="input-sm search-query" data-filter-table="dt-ajax-array"><button class="btn btn-icon"><i class="icon-search"></i></button>
  </div>
</div>
<table cellpadding="0" cellspacing="0" border="0" class="table table-bordered" data-table-name="dt-col-hiding-example"></table>

Script:

//assume 'datagrids/datagrids-col-vis' is on the page...
$('table[data-table-name="dt-col-hiding-example"]').basicdatagrid({
  'aoColumns': [
    {'sTitle': 'Rendering engine', 'mData': 'engine', 'sClass': 'essential' },
    {'sTitle': 'Browser', 'mData': 'browser', 'sClass': 'essential' },
    {'sTitle': 'Platform(s)', 'mData': 'platform', 'sClass': 'optional' },
    {'sTitle': 'Engine version', 'mData': 'version' },
    {'sTitle': 'CSS grade', 'mData': 'grade', 'sClass': 'essential' }
  ],
  'sAjaxSource': '../assets/data/ao.browsers.json',
  'useFloater': false,
  "plugins": ["C"], //enable the Column Vis plugin (assumes datagrids/datagrids-col-vis.js is on the page)
  "oColVis": { //configure the column vis plugin
    "aiExclude": [ 1 ], // don't allow 2nd column to be hidden
    "colVisDropdownSelector": ".col-vis-dropdown-container" //put the column hiding control inside the element at this selector
  }
});

Notes:

  • We forked the DataTables project's "ColVis" plugin for this functionality.
  • You must include the plugin library "[datagrids root]/js/datagrids/datagrids-col-vis.js" on the page. The requirejs id for this in HDX is 'datagrids/datagrids-col-vis'
  • Configure according to the original plug-in's documentation with the following changes.
    • Datatable "plugins" have been moved out of the "sDom" settings parameter into a separate "plugins" settings Array parameter, so enable the plugin by setting 'plugins': ["C"] on your table
    • Place the column hide dropdown inside any element consistent with design guidelines by specifying a "colVisDropdownMenuSelector", which is a valid JQuery selector.
    • The notion of "column groups" to show/hide has been removed
Script Requirements:
  • datagrids
  • datagrids/datagrids-col-vis
