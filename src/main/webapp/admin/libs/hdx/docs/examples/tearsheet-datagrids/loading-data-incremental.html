Data loading: Infinite/Incremental Scrolling

A Datagrid loading many records with infinite/incremental scroll.

Be sure to turn off the HDX's custom pagination with sPagination: null when using this feature

For more information on this Datagrids plugin, see the scroller documentation

Example
Usage

Markup:

<table cellpadding="0" cellspacing="0" border="0" class="table table-bordered" data-table-name="dt-incremental"></table>

Script:

// Incremental Loading/Infinite Scroll
$('table[data-table-name="dt-incremental"]').iidsBasicDataGrid({
    useFloater: false, //turn off cell filtering layer
    "plugins": ["S"],
    "sScrollY": "200px",
    "bDeferRender": true,
    'sPaginationType': null,
    'aoColumns': [
        { 'sTitle': 'Id', 'mData': 'id'},
        { 'sTitle': 'Fulcrum version', 'mData': 'fulcrumVersion'},
        { 'sTitle': 'Platform(s)', 'mData': 'platform'},
        { 'sTitle': 'Differential version', 'mData': 'differentialVersion'},
        { 'sTitle': 'Grade', 'mData': 'grade'}
    ],
    'sAjaxSource': '../assets/sample-data/ao.browsers.11000.json'
});
