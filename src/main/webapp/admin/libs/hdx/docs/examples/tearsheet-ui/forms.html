Form Grid Sizes

Add a div with .col-md-* classes from the grid system for form control sizes. '*' can have a value from 1 to 12.

<div class="col-md-3">
	<input class="form-control" type="text">
</div >

<div class="col-md-6">
	<select class="form-control">
		<option>1</option>
		<option>2</option>
		<option>3</option>
		<option>4</option>
		<option>5</option>
	</select>
</div>

Append and Prepend

Use the .input-group class to wrap form controls with add-on buttons.

<div class="input-group">
	<input class="form-control" type="text">
	<span class="input-group-btn">
		<button type="submit" class="btn btn-icon">
			<i class="icon-search"></i>
		</button>
	</span>
</div>

Apply .input-group-addon as a wrapper to text-only add-ons.

<div class="input-group">
	<span class="input-group-addon"">@</span>
	<input class="form-control" type="text">
</div>

Inline Form

Add .form-inline to finesse the vertical alignment and spacing of form controls.

<form class="form-inline">
<div class="form-group">
  <input type="text" class="input form-control" placeholder="Email">
  <input type="password" class="input form-control" placeholder="Password">
  <button type="submit" class="btn btn-default">Sign in</button>
  </div>
</form>
Horizontal Form

Add .form-horizontal to align all controls and labels horizontally.

Horizontal forms require the wrappers .control-group, .control-label, and .controls for proper alignment and styling.

<form onsubmit="return false;" class="form-horizontal">
	<div class="form-group">
		<label class="control-label col-md-4" for="input01">Text Label</label>
		<div class="col-md-4">
			<input type="text" class="input  controls form-control" id="input01">
		</div>
	</div>
	<div class="form-group">
		<label class="control-label col-md-4" for="select01" >Select Label</label>
		<div class="controls col-md-4">
			<select id="select01" class="form-control" >
				<option>1</option>
				<option>2</option>
				<option>3</option>
				<option>4</option>
				<option>5</option>
			</select>
		</div>
	</div>
	<div class="form-actions" style="padding-left:110px;">
		<button class="btn btn-default">Cancel</button>
		<button type="submit" class="btn btn-primary">Save</button>
	</div>
</form>
