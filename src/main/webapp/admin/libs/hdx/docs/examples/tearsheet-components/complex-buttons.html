Example Description Usage
Stateful

Markup:

<button data-loading-text="loading..." class="btn btn-primary stateful">
Loading State</button>

Script:

$(…).button('loading');

Sets button state to loading—disables button and swaps text to loading text. Loading text should be defined on the button element using the data attribute data-loading-text.

NOTE: Firefox persists the disabled state across page loads. A workaround for this is to use autocomplete="off".

$(…).button('reset');

Resets button state—swaps text to original text.

Script Requirements:
  • bootstrap/bootstrap-button
Toggle

Markup:

<button class="btn btn-primary" data-toggle="button">Single Toggle</button>

Script:

$(…).button('toggle');

Toggles push state. Gives the button the appearance that it has been activated.

The data-toggle attribute enables auto-toggling.

Script Requirements:
  • bootstrap/bootstrap-button

 Button Dropdowns

Use any button to trigger a dropdown menu by placing it within a .btn-group and providing the proper menu markup.

To quickly add dropdown functionality to any element just add data-toggle="dropdown" and any valid bootstrap dropdown will automatically be activated.

<div class="btn-group">
  <button class="btn dropdown-toggle" data-toggle="dropdown">Action <i class="icon-chevron-down"></i></button>
  <ul class="dropdown-menu">
    <li><a href="#" data-toggle="dropdown" >Action</a></li>
    <li><a href="#" data-toggle="dropdown" >Another action</a></li>
    <li><a href="#" data-toggle="dropdown" >Something else here</a></li>
    <li class="divider"></li>
    <li><a href="#" data-toggle="dropdown" >Separated link</a></li>
  </ul>
</div>

You may optionally target a specific dropdown by using data-target="#id" or href="#fat".

NOTE: In some cases—like mobile—dropdown menus will extend outside the viewport. You need to resolve the alignment manually or with custom javascript.

Script Requirements:
  • bootstrap/bootstrap-dropdown
  • Action
  • Another action
  • Something else here
  • Separated link
 Button Dropups

Add .dropup to a .btn-group containing a dropdown to switch to a dropup.

<div class="btn-group dropup">
  <button class="btn dropdown-toggle" data-toggle="dropdown">Action <i class="icon-chevron-up"></i></button>
  <ul class="dropdown-menu">
    <li><a href="#" data-toggle="dropdown" >Action</a></li>
    <li><a href="#" data-toggle="dropdown" >Another action</a></li>
    <li><a href="#" data-toggle="dropdown" >Something else here</a></li>
    <li class="divider"></li>
    <li><a href="#" data-toggle="dropdown" >Separated link</a></li>
  </ul>
</div>

NOTE: If using the original .caret markup from Bootstrap, the .caret will flip automatically for dropups. Otherwise, change the .dropdown-toggle icon to .icon-chevron-up.

 Icon Buttons

<button class="btn btn-primary btn-icon dropdown-toggle" data-toggle="dropdown"><i class="icon-user"></i> <i class="icon-chevron-down"></i></button>

Filter Label Buttons

Add .btn-filter to a .btn-group to trigger special filter alignment rules.

<div class="btn-group btn-filter">
	<a href="#" class="btn btn-default">Asset 01</a>
	<a href="#" class="btn btn-remove"><i class="icon-remove"></i></a>
</div>

Icons with the prefix .color- specify colors for use within filter buttons. The primary GE Charting Palette is available for usage: cyan, green, orange, purple and red. 

<div class="btn-group btn-filter">
	<a href="#" class="btn btn-default"><i class="icon-star color-red"></i>Asset 01</a>
	<a href="#" class="btn btn-remove"><i class="icon-remove"></i></a>
</div>
