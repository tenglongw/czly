Description
Used to expand and collapse container modules.
Usage

Markup:

<section class="module collapsible">
  <header class="module-header">Header</header>
  <div class="module-body collapse in">Body</div>
  <footer class="module-footer collapse in">Footer</footer>
</section>

Script:

<script type="text/javascript">
  require(['module'], function() {
    $('.module.collapsible').collapsible();
  });
</script>

Notes:

  • Make sure to include a collapse class on any element you wish to toggle.
  • If you want an element to be shown by default include an in class, as in the markup example above.
Script Requirements:
  • modules
