Default Buttons

Example Description Usage
Stateful

Markup:

<button data-loading-text="loading..." class="btn btn-primary stateful">
Loading State</button>

Script:

$(…).button('loading');

Sets button state to loading—disables button and swaps text to loading text. Loading text should be defined on the button element using the data attribute data-loading-text.

NOTE: Firefox persists the disabled state across page loads. A workaround for this is to use autocomplete="off".

$(…).button('reset');

Resets button state—swaps text to original text.

Script Requirements:
  • bootstrap/bootstrap-button
Toggle

Markup:

<button class="btn btn-primary" data-toggle="button">Single Toggle</button>

Script:

$(…).button('toggle');

Toggles push state. Gives the button the appearance that it has been activated.

The data-toggle attribute enables auto-toggling.

Script Requirements:
  • bootstrap/bootstrap-button

 Button Dropdowns

Use any button to trigger a dropdown menu by placing it within a .btn-group and providing the proper menu markup.

To quickly add dropdown functionality to any element just add data-toggle="dropdown" and any valid bootstrap dropdown will automatically be activated.

<div class="btn-group">
  <button class="btn dropdown-toggle" data-toggle="dropdown">Action <i class="icon-chevron-down"></i></button>
  <ul class="dropdown-menu">
    <li><a href="#" data-toggle="dropdown" >Action</a></li>
    <li><a href="#" data-toggle="dropdown" >Another action</a></li>
    <li><a href="#" data-toggle="dropdown" >Something else here</a></li>
    <li class="divider"></li>
    <li><a href="#" data-toggle="dropdown" >Separated link</a></li>
  </ul>
</div>

You may optionally target a specific dropdown by using data-target="#id" or href="#fat".

NOTE: In some cases—like mobile—dropdown menus will extend outside the viewport. You need to resolve the alignment manually or with custom javascript.

 Basic Tabs

Markup:

<ul class="nav nav-tabs">
  <li class="active"><a href="#tab1" data-toggle="tab">Section 1</a></li>
  <li><a href="#tab2" data-toggle="tab">Section 2</a></li>

</ul>

Activate tab navigation without javascript by specifying data-toggle="tab" on an element. Adding the nav and nav-tabs classes to the tab ul will apply the bootstrap tab styling.

To align nav links, use the .pull-left or .pull-right utility classes. Both classes will add a CSS float in the specified direction.

Script:

$('#myTab a').click(function (e) {
  e.preventDefault();
  $(this).tab('show');
})

Description

An inspector is used to display detailed information about any particular cell in a data grid. It is inserted as an expanded row in the table below the relevant row. Clicking the cell a second time closes the inspector. Single or multiple inspectors per table are supported as a configurable option.

Usage

<table data-table-name="myTable"  class="table"  >
//add table headers and rows here.
</table>

Script:

require(['jquery','inspector','modules', 'ge-bootstrap', 'datagrids' ],function ($, inspector) {
	
	$(function() {
      	var table = $('table[data-table-name="myTable"]').iidsBasicDataGrid({
        useFloater: false,
        isResponsive: true,
        bInfo: false,
        bSortClasses: false,
      	bPaginate: false,
      	fnCreatedRow: function fnCreatedRow(nRow) {
       		  $('td',nRow).addClass("inspectable");
	       	  $('td', nRow).click(function() {
	      			var inspectorContent= HTML content to display
	      			var allowMultipleInspectors = true/false;
	      			//This will be called per cell click
	      			$(this).inspector(nRow, tbl ,$(inspectorContent), allowMultipleInspectors);
	          });
        }
      });
      // fetch the (hungarian) table for later use
      var tbl = table.dataTable();
    });
  });
