Example Description Usage
This class is a widget for selecting a date. Focus on the input (click, or use the tab key) to open an interactive calendar in a small overlay. Choose a date, click elsewhere on the page (blur the input), or hit the Esc key to close. 
<div class="input-append bootstrap-datepicker">
	<input type="text" class="form-control input-sm datepicker">
	<span class="input-group-btn">
		<span class="add-on">
			<button class="btn btn-default" type="button">
				<i class="icon-calendar"></i>
			</button>
		</span>
	</span>
</div>
