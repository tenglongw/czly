 Inspector List (Accordion) 
    <div class="panel-group" id="accordion">
        <div class="panel panel-default">
            <div class="panel-heading">
                <a class="panel-toggle collapsed" data-toggle="collapse" data-parent="#accordion" href="#collapseOne"><i class="icon-caret-right"></i>Collapsible Group Item #1</a>
            </div>
            <div id="collapseOne" class="panel-collapse collapse">
                <div class="panel-body">collapse content goes here</div>
            </div>
        </div>
    </div>

Just add data-toggle="collapse" and a data-target to element to automatically assign control of a collapsible element. The data-target attribute accepts a css selector to apply the collapse to. Be sure to add the class collapse to the collapsible element and collapsed to the header link. If you'd like it to default open, add the additional class in to the collapse body, and remove collapsed from the header.

NOTE: See Bootstrap documentation for a full list of options, methods and events for controlling collapsibles.

Script Requirements:
  • bootstrap/bootstrap-collapse
  • bootstrap/bootstrap-transition
