Examples
Description

The contextual horizontal stacked bar also known as the pill in the data-visualizations package.
Usage

Markup:

 <div class="module col-sm-6 col-md-6" id="pill-viz" >
	 <div class="module-body" >
		<div data-visualization="pill" data-values="1,1,1" data-label="Total Lorem Ipsum" data-classes="success,warning,danger" data-thickness="9"/>
		<id="single" data-visualization="pill" data-values="1" data-label="Total Lorem Ipsum" data-classes="success,warning,danger" data-thickness="9"/>
		<id="double" data-visualization="pill" data-values="1,1,0" data-label="Total Lorem Ipsum" data-classes="success,warning,danger" data-thickness="9"/>
	 </div>
  </div>

Script:

require(['hdx-declarative-visualizations'], function(visualizations) {
  // Initialize Bar Visualizations
  $('div[data-visualization="pill"]').each(function(i, element) {
    var pill = visualizations.pill(element);
  });
});
Script requirements:
  • hdx-declarative-visualizations
