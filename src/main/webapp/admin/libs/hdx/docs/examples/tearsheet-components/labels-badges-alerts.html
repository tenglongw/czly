Reset Links 		Alerts 
<div class="alert alert-warning">
  <strong>Warning!</strong> Best check yo self.
</div>

Use one of .alert-success, .alert-info or .alert-error to colorize the .alert for the appropriate context.

<div class="alert alert-success">
  <strong>Well done!</strong> You successfully read this.
</div>

Add .alert-fixed-top to .alert to fix the position to the top of the browser window and only round the bottom corners.

Add an optional button before the message. Use the attribute data-dismiss="alert" to automatically enable alert dismissal.

Markup:

<div class="alert alert-success">
  <button type="button" class="close" data-dismiss="alert">×</button>
  <strong>Well Done!</strong> You successfully read this.
</div>

Script:

$(…).alert('close');
Script Requirements:
  • bootstrap/bootstrap-alert
