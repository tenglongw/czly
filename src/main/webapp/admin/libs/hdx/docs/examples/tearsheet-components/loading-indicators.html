Example Description Usage
Progress Bar (Basic) 
<div class="progress">
  <div class="progress-bar" style="width: 45%;"></div>
</div>

Use a percentage width on .progress-bar to advance progress bars.

Add one of .progress-success, .progress-info, .progress-warning or .progress-danger to colorize .progress according to context.

<div class="progress progress-success">
  <div class="progress-bar" style="width: 45%;"></div>
</div>
Progress Bar (Striped)

Add .progress-striped to add a striping effect to the progress bar.

<div class="progress progress-striped">
  <div class="progress-bar" style="width: 45%;"></div>
</div>
Progress Bar (Animated)

Add .active to a striped progress bar to trigger animation.

<div class="progress progress-striped active">
  <div class="progress-bar" style="width: 45%;"></div>
</div>
Activity Indicator (Spinner)

Markup:

<span class="spinner"></span>

Script:

$(…).spin();

The size and color of the spinner can be modified by passing options when initializing.

NOTE: The options listed in the example are default.

var options = {
  // The number of lines to draw
  lines: 10,
  // The length of each line
  length: 4,
  // The line thickness
  width: 2,
  // The radius of the inner circle
  radius: 4,
  // The rotation offset
  rotate: 0,
  // #rgb or #rrggbb
  color: '#313337',
  // Rounds per second
  speed: 1.2,
  // Afterglow percentage
  trail: 60,
  // Whether to render a shadow
  shadow: false,
  // Whether to use hardware acceleration
  hwaccel: false,
  // The CSS class to assign to the spinner
  className: 'spinner',
  // The z-index (defaults to 2000000000)
  zIndex: 2e9,
  // Top position relative to parent in px
  top: 'auto',
  // Left position relative to parent in px
  left: 'auto'
};
$(…).spin(options);

NOTE: See spin.js documentation for a full list of configuration options.

Script Requirements:
  • spinner
