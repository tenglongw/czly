Row grouping

A Datagrid using interactive row grouping.

Example
Usage

Markup:

<div class="row-grouping-dropdown-container"></div>
<table cellpadding="0" cellspacing="0" border="0" class="table table-bordered" data-table-name="dt-row-grouping-example"></table>

Script:

//assume 'datagrids/datagrids-row-grouping' is on the page...
$('table[data-table-name="dt-row-grouping-example"]').basicdatagrid({
  'aoColumns': [
    {'sTitle': 'Rendering engine', 'mData': 'engine', 'sClass': 'essential'},
    {'sTitle': 'Browser', 'mData': 'browser', 'sClass': 'essential'},
    {'sTitle': 'Platform(s)', 'mData': 'platform', 'sClass': 'optional'},
    {'sTitle': 'Engine version', 'mData': 'version'},
    {'sTitle': 'CSS grade', 'mData': 'grade', 'sClass': 'essential'}
  ],
  'sAjaxSource': '../assets/data/ao.browsers.json',
  'useFloater': false,
  'plugins': ["G"], //enable the row grouping plugin (assumes datagrids/datagrids-row-grouping.js is on the page)
  "bPaginate": false, // pagination is not compatible with row grouping
  "rowGrouping": { //configure the row grpuping plugin
    rowGroupingDropdownSelector: ".row-grouping-dropdown-container" //tell the plugin where to put the row grouping control
  }
});

Notes:

  • You must include the plugin library "[datagrids root]/js/datagrids/datagrids-row-grouping.js" on the page. The requirejs id for this in HDX is 'datagrids/datagrids-row-grouping'
  • Datatable "plugins" have been moved out of the "sDom" settings parameter into a separate "plugins" settings Array parameter, so enable the plugin by setting 'plugins': ["G"] on your table
  • Place the row grouping dropdown inside any element consistent with design guidelines by specifying a "rowGroupingDropdownSelector", which is a valid JQuery selector.
  • Pagination must be disabled when row grouping is enabled
Script Requirements:
  • datagrids
  • datagrids/datagrids-row-grouping
